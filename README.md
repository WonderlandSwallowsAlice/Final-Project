# CS208 Final Project

This is a simple website for a Downtown Donut Coffee Shop. It was built using sql, pug, css and javascript languages. To run the code you first need to install MariaDB by running ./setup_scripts/install_db.sh command in terminal. You then have to run sudo mysql -u root -p < ./setup_scripts/create_demo_table.sql to be able to leave and read the reviews. I also had to run this command a few times npm install escape-html, so if there are any difficulties or errors during launching it in browser, try that as well. Then just run npm start and the program will open in browser. You can navigate to Home, Menu, About Us and Reviews pages. Pressing menu will open a new pages, where you can view menu in pdf format. In reviews you can leave a review yourself and view the reviews left. 

You cannot leave name or comment sections empty or exceed the maximum amount of character specified, else you will get an error message. If you click on doodash or uber words in home, you will be redirected to these websites.
